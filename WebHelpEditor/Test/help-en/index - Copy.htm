Welcome to AQUARIUS! 

testAQUARIUS Time-Series Software is the premier platform for managing environmental data.  Both water flow and water quality data can be stored in the system, along with supporting information from meteorological and many other sensors.

AQUARIUS Server is the core of the AQUARIUS Time-Series Software solution.  AQUARIUS Server provides centralized, automated, secure, and scalable data management.  It connects to a relational database (either Oracle or SQL Server) that houses an open and easily understood data schema.  AQUARIUS Time-Series can be any size (up to millions of points).  Data can be stored for any number of parameter types for any number of measurement locations.

Aquatic Informatics is continually improving and extending the capabilities of the AQUARIUS Software platform, incorporating feedback from users and responding to changes and trends in environmental science.

What's New in AQUARIUS 3.4?

AQUARIUS version 3.4 includes the accumulated improvements and features from previous versions as well as several new exciting features.

Ratings Based on the Index Velocity Method

AQUARIUS now supports ratings based on the Index Velocity method. This new feature is available as a beta release to a limited number of customers. Contact your Aquatic Informatics support representative if you wish to be part of the beta program.

Improvements in Springboard Load Time

Springboard now loads more quickly after launch. The degree of improvement depends on the number of locations and the network speed, but on average, a 2X increase may be expected.

ADCP Discharge Measurements

Improved support for importing ADCP Discharge measurements from files. WinRiver II and SonTek files are now parsed for additional information when being imported via Field Visit Tool. The details are output in the Discharge Measurement summary report.

Data Correction Toolbox Surrogate Signals

Data Correction Toolbox now supports the ability to add surrogate signals after launch the toolbox. Signals may be selected from the current location, or browsed from a different location. Use the option "File -> Select Data Sets" to enable this feature. Note that this feature is not supported on computers which have Internet Explorer 8 or lower.

Web-based Administration for Generic Data Provider

New web-based administration pages to control and monitor Generic Data Provider synchronization. Consult the AQUARIUS Generic Data Provider document for further information.
