  Analysis & Remarks

The Analysis & Remarks tab allows the user to enter comments, remarks or descriptions for a specific location and its associated data sets.

The remarks can be classified by date ranges of applicability, type or both. The remarks can be sorted by clicking on any of the column headers in the table. The text of a remark can be formatted using the WYSIWYG Editor ("What You See Is What You Get" Editor).
