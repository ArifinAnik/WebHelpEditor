  User Access Tab

The User Access tab allows a User with the correct privileges ("Can Edit Location Details" privileges) to define User Access for specific locations.

A Default User Role can be applied for the selected location; this will override all users in the system and grant them the selected role. Individual Users may also be selected and given a role for the selected location. This new role will override the one granted at the system level by the AQUARIUS Manager.

For Water Survey Canada, the User Access tab is a non-editable, read-only list of users populated from the HYDEX database.



The user access tab allows the administrator to perform three tasks:

   Show Effective Roles - The first task available is to show a list of all users in the system and what their respective role is at the current location.


Click on the Show effective roles" button, to view all users in the system along with their assigned roles at the current location. Both columns are sortable

   Default User Role - The second task available is to assign a default role for the current location. To specify a default role for all users at the current location pick a role from the drop down list labeled Default User Role. 

   Specific User Roles - The third task available is to manage user specific roles for the location. This will override the role given to the user at the system level by the AQUARIUS Manager.

To grant a user a location specific role click on the "+" button. This action will bring up a popup window where you can select one or several users and grant them a role from the dropdown list. To select/deselect a user in the list, click on the username.

To edit a user's location specific role, select one or more rows from the table labeled Specific User Roles. Right click after the selection is made and choose the Set Role" option from the context menu. A pop-up window will appear allowing the user to set a new role for the selected rows. To edit a single user's role, click on the role and select a different one from the dropdown list.

To remove a user's location role click on the "-" button. This will remove the location specific role only, if the user has a role at the System level, the user will continue to have this role at the specified location.
