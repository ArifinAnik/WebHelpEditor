AQUARIUS Help Manual
  General Tab

The General tab allows the user to view the details associated with the selected location including:

   Location name
   A unique identifier
   Latitude and Longitude degrees
   As well as a whole host of other additional location specific attributes

The icons in the general tab provide the following functions:

New Folder - Allows the user to create a new folder under the parent location. The name can be amended by clicking onto the newly created folder.

New File - Allows the user to add a new file to that selected folder. Any file can be added to the folder or parent such as photos, reports, .pdf or data files associated with that location.

Delete File/Folder - Allows the user to delete any file or folder that is available. Select the File/Folder of choice and click on the Delete File/Folder icon. It will be immediately deleted.

Set Location Photo - Allows the user to set a photo for the location. Select and add the new photo and click the Set Location Photo icon to set the photo for the location. The image will upload and appear in the blank section below the General Tab

  Download File - Allows the user to download the selected file to their computer
